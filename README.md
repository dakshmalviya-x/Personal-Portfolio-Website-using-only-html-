# Personal Portfolio Website

## Project Overview
This project is a simple HTML portfolio website created to showcase personal information, skills, and contact details.

## Features
- Semantic HTML5 structure
- About section
- Skills section
- Contact form with validation
- Internal navigation links
- Profile image
- External links to GitHub and LinkedIn

## Folder Structure

Portfolio-Website/
│
├── index.html
├── README.md
├── images/
│     └── profile.jpg
└── requirements.txt

## Concepts Learned
- HTML5 document structure
- Semantic tags
- Headings and paragraphs
- Lists (ul and ol)
- Forms and validation
- Images and links
- Internal navigation

## Testing
- Tested in Google Chrome
- Validated using W3C HTML Validator
